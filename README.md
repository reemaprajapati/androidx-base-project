# androidx-base-project

Android Base Project Setup with AndroidX (Android Jetpack).

### Components:
1. [Android KTX](https://developer.android.com/kotlin/ktx)
2. [Data Binding](https://developer.android.com/topic/libraries/data-binding/)
3. [Lifecycle](https://developer.android.com/topic/libraries/architecture/lifecycle)
4. [LiveData](https://developer.android.com/topic/libraries/architecture/livedata)
5. [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel)
6. [Room](https://developer.android.com/topic/libraries/architecture/room)
7. [Permissions](https://developer.android.com/guide/topics/permissions/overview)


### Libraries
1. [Retrofit](https://square.github.io/retrofit/)
2. [Dagger2](https://google.github.io/dagger/)
3. [RxJava/RxKoltin](https://github.com/ReactiveX)
4. [Stetho](http://facebook.github.io/stetho/)
5. [Glide](https://github.com/bumptech/glide)
6. [LeakCanary](https://github.com/square/leakcanary)
